Heart Disease Prediction

An AI-powered Machine Learning application for predicting the risk of heart disease using patient medical information & interactive Power BI dashboard

📌Project Overview

Heart Disease Prediction is a Machine Learning project that predicts whether a patient is likely to have heart disease based on clinical and health-related features. The project also includes interactive Power BI dashboards to analyze patient health data and support better decision-making.

🚀Features

Predict heart disease risk

Machine Learning-based prediction

Interactive Power BI dashboards

Patient health analysis

-Easy-to-use and accurate prediction system
